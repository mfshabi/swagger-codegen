reportsByZoneRecordBrowsers

Get reports of zone browsers

URL

https://api.5centscdn.com/v1/zones/reports/http/record/{zoneid}/on/browsers

HTTP Method

POST

Response Type

inline_response_200_53

Parameters

  • Header: X-API-Key String

    Your API Key

  • Path: zoneid String

    Zone ID

  • Form: frequency String

  • Form: start_date String

  • Form: end_date String
