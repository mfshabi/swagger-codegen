PostZoneEdge

  • compression : String
    The values can be Y or N.<br> Set to Y if edge compression should be enabled.<br> Set to N if edge compression should be disabled.
  • compressionlevel : String
    Set the value of edge[compressionlevel] when edge[compression] is enabled. <br>The values can be 1, 2, 3, 4, 5, 6.
  • disposition : String
    Values can be Y or N<br> Set to Y to enable edge disposition. <br> Set to N to disable edge disposition.
  • stripcookie : String
    Values can be Y or N<br> Set to Y to enable. <br> Set to N to disable.
  • xff : String
    Values can be Y or N<br> Set to Y to enable. <br> Set to N to disable.
  • cors : String
    Values can be Y or N<br> Set to Y to enable. <br> Set to N to disable.
  • webp : String
    Values can be Y or N.<br> Set to Y to enable WebP.<br> Set to N to disable WebP.
  • pseudostreaming : String
    Values can be Y or N.<br> Set to Y to enable pseudo stream seeking.<br> Set to N to disable pseudo stream seeking.
