Streamsrestreamnew_record

  • enabled : String
    Values can be Y or N.<br> Set Y for enable DVR<br> Set N for Diasable DVR<br>Note: Applicable only for users with DVR
  • retention : String
    To specify the Retention period.<br>A digit followed by letter ‘d’: Retention in days, min: 1, max: 14 eg: 7d (7 days of DVR) <br/>Digit followed by letter ‘h’: Retention in hours, min: 2, max: 336<br> Eg: 6h (6 hrs of DVR)
