UpdatePushStream

  • server : String
    Server code of the publish stream which is to be updated.<br>Server code should be one of the server code from the GET /streams/push/servers call
  • audioonly : String
    The values can be Y or N. <br> Set to Y if the HLS audio only stream should be added to the manifest file
  • backup : String
    URL, A link to backup video to roll, when there is no publish
