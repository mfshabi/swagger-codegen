Putplaylist

  • name : String
    Playlist name.<br>Name should have minimum 2 chars, and maximum 32 chars length.
  • type : String
    Values can be 'live' or 'file'.
  • length : String
    Length should be in the format hh:mm:ss. length would be set if type is 'live'.
  • schedule : String
    The values can be 'instant', ' scheduleontime' or 'scheduleondatetime'
  • scheduleontime : String
    Scheduleontime should be in the format of hh:mm:ss. <br>It can be use if the 'schedule' is 'scheduleontime'.
  • scheduleondatetime : String
    Scheduleondatetime's format should be yyyy-mm-dd hh:min:sec . <br>It can be used if 'schedule' is 'scheduleondatetime'.
