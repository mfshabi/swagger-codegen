PullServerToken

  • policy : String
    Values can be Y or N. <br>Set to Y if secure token should be enabled.<br> Set to N if secure token should be disabled.
  • list : String
    List can be a random alpha numeric string, [A-Za-z0-9]
  • ips : String
    List of IPs to whitelist when secure token is enabled. IPs should be separated by comma. <br>Eg: 192.168.0.1, 192.168.0.2
  • timeout : String
    Secure token link expiration time in seconds min: 5, max: 25200
