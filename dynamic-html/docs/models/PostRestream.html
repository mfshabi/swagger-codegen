PostRestream

  • _METHOD : String
    request method PUT
  • name : String
    Stream name.<br>Should be in the format part-1/part-2. Each part should have minimum 3 chars, separated by forward slash ‘/’, without any space
  • restreamtype : String
    The values can be mediaserver or ffmpeg.<br> mediaserver: for restreaming via media server.<br> ffmpeg: for restreaming via ffmpeg server
  • server : streamsrestreamnew_server
  • url : streamsrestreamnew_url
  • audioonly : String
    Values can be Y or N<br>Set to Y if the HLS audio only stream should be added to the manifest file
  • domainlock : streamstimeshiftnew_domainlock
  • geoblock : streamsrestreamnew_geoblock
  • backup : String
    A link to backup video to roll when there is no publish
  • audiofix : String
    Used to transcode the audio stream to aac using ffmpeg if required.<br> Available only when 'ffmpeg' restream type is chosen.
  • securetoken : streamstimeshiftnew_securetoken
  • record : streamsrestreamnew_record
  • eac : streamsabrnew_eac
