Streamstimeshiftnew_securetoken

  • policy : String
    Values Can be Y or N. <br> Set to Y if secure token should be enabled<br> Set to Y if Secure token should be disabled.
  • list : String
    The value should be random alpha numeric string, [A-Za-z0-9]
  • ips : String
    List of IPs to whitelist when geoblock is enabled. <br> IPs should be seperated by comma. <br>Eg: 192.168.0.1, 192.168.0.2
  • timeout : Integer
    Secure token link expiration time in seconds min: 5, max: 25200
