Streamsabrnew_domainlock

  • policy : String
    Values can be Y or N. <br>Set to Y if domainlock should be enabled.<br> Set to N if domainlock should be disabled.
  • list : String
    List of domains seperated by comma .<br> Eg: www.domain-one.com, www.domain-two.com
  • ips : String
    Comma separated list of IPs to whitelist when domain lock is enabled.<br> eg: 192.168.0.1, 192.168.0.2
  • type : String
    The values can be whitelist or blacklist.
