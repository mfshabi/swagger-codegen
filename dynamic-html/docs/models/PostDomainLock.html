PostDomainLock

  • policy : String
    Values can be Y or N.<br>Set to Y if domainlock should be enabled.<br> Set to N if domainlock should not to be enabled.
  • list : String
    List of domains seperated by comma.<br>Eg: www.domain-one.com, www.domain-two.com
  • ips : String
    List of IPs to whitelist when domain lock is enabled. IPs should be separated by comma. <br>Eg: 192.168.0.1, 192.168.0.2
  • type : String
    The values can be whitelist or blacklist
